# HTMLbase
Base files for a quick HTML project.
Find/Replace "HTMLbase" in index.html and give it your project name.

```
HTMLbase/
├── css/
│   ├── bootstrap.css
│   ├── style.css
└── fonts/
│   └── ...
└── images/
│   └── favicon
│       └── ...
└── js/
│   ├── bootstrap.js
│   └── jquery-3.1.1.min.js
└── index.html
```
